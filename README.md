# SteemApi

### How To Use (Rails 4+)
- Add Gem to Gemfile
	*	`gem 'steem_api', '~> 1.0'`
- Bundle Install Gems
	* `bundle install`

### How To Use (Standalone)
- `gem install steem_api`
- `irb`
- `require 'steem_api'`
- `SteemApi::Comment.last`

### Models
- Account
- Block
- Comment
- Token
- Transaction

- Fork this repo
